# 🚗 Parking Project

## Overview

Machine learning-based parking allocation system that predicts parking duration and assigns slots dynamically.

## Tech Stack

* Python
* SQLite
* Machine Learning

## Features

* Slot allocation based on vehicle type & size
* Predictive parking duration
* Real-time database updates

## How to Run

1. Run setup_database.py
2. Run train_model.py
3. Run parking_engine.py
